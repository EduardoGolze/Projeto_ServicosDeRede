# Projeto de Infraestrutura de Redes: TechNet Solutions

Este repositório contém a documentação completa da implementação da infraestrutura de servidores da empresa **TechNet Solutions**, utilizando o sistema operacional Linux (Ubuntu Server).

## Estrutura do Projeto

O projeto foi dividido em quatro etapas fundamentais, documentadas individualmente nos arquivos abaixo:

1. **[Etapa 1: Configuração de Rede e Internet](rede_internet.md)** - Configuração de interfaces e testes de conectividade.
2. **[Etapa 2: Servidor de Arquivos (Samba)](servidor_arquivos.md)** - Compartilhamento de arquivos e controle de permissões.
3. **[Etapa 3: Terminal Server (XRDP)](terminal_server.md)** - Acesso remoto gráfico simultâneo.
4. **[Etapa 4: Autenticação Centralizada (OpenLDAP)](openldap.md)** - Criação da árvore de diretórios e cadastro de usuários.

---

## Conclusão do Projeto
A implementação atingiu plenamente todos os requisitos estipulados. A empresa TechNet Solutions passa a contar com um ambiente dotado de endereçamento estável, compartilhamento de arquivos seguro, capacidade de gerenciamento remoto simultâneo e uma base de autenticação centralizada e escalável via OpenLDAP, alinhada com as melhores práticas de mercado.
